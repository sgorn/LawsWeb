Dictionnaire des données

La liste des lois et règlements codifiés en XML contient les éléments suivants :

ActsRegsList
Un élément-balise qui englobe toute la liste.
Acts
Un élément-balise qui englobe toutes les lois codifiées.
Act
Une loi fédérale codifiée disponible sur le site Web lois-laws.justice.gc.ca.
UniqueID
Un code alphanumérique unique utilisé pour identifier une partie spécifique de la législation.
Language
Un indicateur de la langue des titres pour cette partie de la législation contenant soit les caractères « eng » pour l’anglais ou « fra » pour le français.
Title
Un titre abrégé pour la législation qui peut être ou non un titre abrégé officiel.
CurrentToDate
La législation comprend toutes les dispositions qui sont en vigueur à partir de la date indiquée dans cet élément.
RegsMadeUnderAct
Un élément-balise dans une loi qui contient une liste de tous les règlements pris en vertu de la loi. Cet élément sera vide si aucun règlement n’a été pris en vertu de la loi.
Reg
Cet élément vide contient un attribut « idref » dont la valeur est l’identifiant d’un règlement dans ce fichier XML.
LinkToXML
Cet élément contient l’URL de la version XML à jour de la législation.
LinkToHTMLToC
Cet élément contient l’URL de la version HTML à jour de la table des matières de la législation.
Regulations
Un élément-balise qui englobe tous les règlements codifiés.
Regulation
Cet élément représente un texte règlementaire spécifique disponible sur le site Web lois-laws.justice.gc.ca. L’attribut « id » dans cet élément est un identifiant unique pour cet élément; l’attribut « olid » est l’identifiant du même texte règlementaire dans la langue opposée.
