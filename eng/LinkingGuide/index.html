How to Create Stable Links

This online resource of Laws is updated regularly. When creating a link to a particular section within a consolidated Act or regulation, it should be done in such a way as to avoid creating broken links when the site is updated. To create stable links, we encourage you to follow the recommendations below.

Using the Consolidated Acts or Consolidated Regulations listings, navigate to the Table of Contents page for the desired law to find the URL. For example the URL for the Access to Information Act is:

https://laws-lois.justice.gc.ca/eng/acts/A-1/index.html

You can create a stable link to the Table of Contents page, a particular section or to the full document.

For example to create a link to section 1 of the Access to Information Act, replace "index.html" with "section-1.html".

https://laws-lois.justice.gc.ca/eng/acts/A-1/section-1.html

To create a link to the full document version of the Access to Information Act, replace the "index.html" with "FullText.html".

https://laws-lois.justice.gc.ca/eng/acts/A-1/FullText.html
