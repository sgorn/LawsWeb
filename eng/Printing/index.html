Printing

PDF Version

To print a document in PDF format, simply click on the PDF link provided. Should the size of a table, form or graphic prevent the document from displaying correctly, please contact us.

HTML Version

Print any page on the Justice Laws Website by using the Print option available in the Internet browser being used. The website automatically removes the menus and web-specific items for the purpose of printing a hard copy from the HTML version.

General

Use your browser's Print Preview function to determine whether the unilingual document fits properly on the page. In cases where text cuts off at the end of a line, try changing the page orientation from portrait to landscape to resolve the problem. Also, Firefox 1.5 (or above) and Internet Explorer 7 (or above) offer a "shrink to fit" function that will attempt to force the document into the space provided.

The Department of Justice has made use of Cascading Style Sheets (CSS) in preparing this site in order to improve access to the consolidated Acts and regulations. Although the use of CSS is the best method for creating fully accessible websites, users with older browsers and printers may experience difficulties.

If you are experiencing difficulties, we recommend that you upgrade your browser version; again, Internet Explorer 7 (or above) and Mozilla Firefox 1.5 (or above) are recommended.
