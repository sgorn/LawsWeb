Data Dictionary

The Consolidated Acts and regulations in XML uses the following XML elements.

ActsRegsList
A wrapper element that goes around the entire list.
Acts
A wrapper element that contains all the consolidated Acts.
Act
A consolidated federal Act available on the laws-lois.justice.gc.ca website.
UniqueID
A unique alphanumeric code used to identify a specific piece of legislation.
Language
An indicator of the language of the titles for this piece of legislation containing either the text string “eng” for English or “fra” for French.
Title
A short title for the legislation that may or may not be an official short title.
CurrentToDate
The legislation includes all provisions that are in force as of the date contained in this element.
RegsMadeUnderAct
A wrapper element in an Act that contains a list of all Regulations that were made under the Act. This element will be empty if no Regulations have been made under the Act.
Reg
This empty element contains an attribute “idref” whose value is the id of a Regulation in this XML file.
LinkToXML
This element contains a URL that points directly to the current legislation in its XML format.
LinkToHTMLToC
This element contains a URL that points directly to the table of contents of the current legislation in HTML.
Regulations
A wrapper element that contains all the consolidated Regulations.
Regulation
This element represents a specific regulatory instrument available on the laws-lois.justice.gc.ca website. The id attribute on this element is a unique id for the element; the “olid” attribute is the id of the same regulatory instrument in the opposite language.
